# Heathcare_Analytics

This project presents a collection of healthcare data analyses exploring patient outcomes, hospital operations, and healthcare costs using publicly available datasets. The goal is to demonstrate how data analysis and machine learning can provide insights into healthcare systems and patient care.
The project consists of three analytical studies implemented in Jupyter notebooks:
Readmission Risk Analysis
Hospital Operations Analysis
Healthcare Cost Analysis
Each notebook examines a different dataset and focuses on a distinct healthcare question.
